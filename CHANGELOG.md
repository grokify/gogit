## [v0.5.0] - 2026-07-18

### Highlights

- Repository renamed from gitscan to gogit: a generic Git base library for Go, with the gitscan CLI preserved at cmd/gitscan
- New core git library: repository discovery, commit-log parsing with Co-authored-by trailers and change stats, branch/origin metadata, tag dates
- Workflow compliance scanning against a reference repository

### Breaking

- Module path changed from github.com/grokify/gitscan to github.com/grokify/gogit; the root package is now the gogit library and the CLI entrypoint moved to cmd/gitscan (install with go install github.com/grokify/gogit/cmd/gitscan@latest) ([`744e28f`](https://github.com/grokify/gogit/commit/744e28f))
- --go-git flag removed from all gitscan subcommands; scanner.GoGitBackend no longer exists and the go-git/v5 dependency tree is dropped from the module ([`3987794`](https://github.com/grokify/gogit/commit/3987794))

### Added

- gogit.Discover: depth-bounded repository discovery that does not descend into repositories ([`2e3097e`](https://github.com/grokify/gogit/commit/2e3097e))
- Repo.Log: commit-log parsing with calendar-date filtering, author/committer identities, subjects, trailers, and --numstat change stats ([`2e3097e`](https://github.com/grokify/gogit/commit/2e3097e))
- Commit.CoAuthors: Co-authored-by trailer extraction to name/email signatures ([`2e3097e`](https://github.com/grokify/gogit/commit/2e3097e))
- Repo.Branch, Repo.OriginURL, and NormalizeRemoteURL (https/ssh/git remotes to host/path identifiers) ([`2e3097e`](https://github.com/grokify/gogit/commit/2e3097e))
- Repo.Tags and Repo.TagsWithDates (tag creation dates via for-each-ref) ([`2e3097e`](https://github.com/grokify/gogit/commit/2e3097e))
- scanner: workflow compliance checking against a reference repo via --check-workflows and --ref-repo ([`756baaf`](https://github.com/grokify/gogit/commit/756baaf))
- since --unpushed/-u flag: filter repos with uncommitted or unpushed changes ([`d219b43`](https://github.com/grokify/gogit/commit/d219b43))

### Changed

- git execution pinned to C locale with credential prompts disabled, making error-condition detection locale-independent ([`2b320c2`](https://github.com/grokify/gogit/commit/2b320c2))
- scanner rebuilt on the root gogit library: IsRepo now also recognizes worktree gitfiles ([`3987794`](https://github.com/grokify/gogit/commit/3987794))
- commit-log parsing refactored to leading-sentinel records, removing lookahead numstat attribution ([`f0273a5`](https://github.com/grokify/gogit/commit/f0273a5))

### Dependencies

- go-git v5.19.1, mogo v0.74.6, yaml.v3 v3.0.1; GitHub Actions checkout/setup-go pinned to v6 ([`744e28f`](https://github.com/grokify/gogit/commit/744e28f))

### Documentation

- README reframed for the gogit library with the gitscan CLI as a bundled tool ([`2485217`](https://github.com/grokify/gogit/commit/2485217))
- Added docs/releases/v0.5.0.md and MkDocs site scaffolding

### Tests

- Core library tests with fixture repositories in isolated git config (no inherited signing, hooks, or templates) ([`816e6b9`](https://github.com/grokify/gogit/commit/816e6b9))
- scanner characterization tests: status detection, module mismatch, replace counting ([`c9a012c`](https://github.com/grokify/gogit/commit/c9a012c))

### Infrastructure

- CI migrated to shared workflow files (go-ci, go-lint, go-sast-codeql) ([`9521180`](https://github.com/grokify/gogit/commit/9521180))

## [v0.4.0] - 2026-02-21

### Highlights

- Restructured CLI into focused Cobra subcommands
- New `since` subcommand with optional `--dep` flag for AND logic
- New `dep` subcommand for dependency filtering
- Homebrew distribution via `brew install gitscan`

### Breaking

- `--since` flag removed from root command; use `gitscan since <duration>` instead ([`ce196b9`](https://github.com/grokify/gogit/commit/ce196b9))
- `--dep` flag removed from root command; use `gitscan dep <module>` instead ([`ce196b9`](https://github.com/grokify/gogit/commit/ce196b9))

### Added

- `since` subcommand: filter repos by modification time with duration argument ([`ce196b9`](https://github.com/grokify/gogit/commit/ce196b9))
- `dep` subcommand: filter repos by dependency on a specific module ([`ce196b9`](https://github.com/grokify/gogit/commit/ce196b9))
- `since --dep` flag: combine time and dependency filters with AND logic ([`ce196b9`](https://github.com/grokify/gogit/commit/ce196b9))
- Shared helpers in `cmd/shared.go`: `resolvePath()`, `createGitBackend()`, `parseDuration()` ([`ce196b9`](https://github.com/grokify/gogit/commit/ce196b9))
- GoReleaser configuration for cross-platform builds (linux, darwin, windows; amd64, arm64) ([`723b458`](https://github.com/grokify/gogit/commit/723b458))
- GitHub Actions release workflow triggered on version tags ([`723b458`](https://github.com/grokify/gogit/commit/723b458))
- Homebrew formula distribution via `grokify/homebrew-tap` ([`723b458`](https://github.com/grokify/gogit/commit/723b458))

### Changed

- Root command now only scans for issues (uncommitted, replace, mismatch) ([`ce196b9`](https://github.com/grokify/gogit/commit/ce196b9))
- Time filtering moved from `--since` flag to `since` subcommand ([`ce196b9`](https://github.com/grokify/gogit/commit/ce196b9))
- Dependency filtering moved from `--dep` flag to `dep` subcommand ([`ce196b9`](https://github.com/grokify/gogit/commit/ce196b9))

### Documentation

- Updated README with Homebrew installation and new subcommand structure
- Added RELEASE_NOTES_v0.4.0.md

## [v0.3.0] - 2026-02-14

### Highlights

- Git backend abstraction with go-git and CLI implementations
- Optional `--go-git` flag for pure Go git operations (no process spawning)

### Added

- Git backend interface with `GitBackend`, `GoGitBackend`, and `CLIGitBackend` types ([`4e2ea58`](https://github.com/grokify/gogit/commit/4e2ea58))
- `--go-git` flag to use go-git library instead of git CLI ([`4e2ea58`](https://github.com/grokify/gogit/commit/4e2ea58))
- Go-git integration for pure Go git operations without external process spawning ([`4e2ea58`](https://github.com/grokify/gogit/commit/4e2ea58))

### Changed

- Git CLI is now the default backend (faster and more compatible) ([`4e2ea58`](https://github.com/grokify/gogit/commit/4e2ea58))
- Scanner package uses `GitBackend` interface for pluggable git implementations ([`4e2ea58`](https://github.com/grokify/gogit/commit/4e2ea58))

### Performance

- Git CLI backend: ~2.5s for 600 repos (default, faster) ([`4e2ea58`](https://github.com/grokify/gogit/commit/4e2ea58))
- Go-git backend: ~10s for 600 repos (optional, pure Go) ([`4e2ea58`](https://github.com/grokify/gogit/commit/4e2ea58))

### Dependencies

- Added `github.com/go-git/go-git/v5` for pure Go git operations ([`4e2ea58`](https://github.com/grokify/gogit/commit/4e2ea58))

### Documentation

- Updated README with `--go-git` flag and backend comparison table ([`f9e8523`](https://github.com/grokify/gogit/commit/f9e8523))

## [v0.2.0] - 2026-02-14

### Highlights

- New `order` subcommand for topological dependency ordering
- Time-based filtering with `-since` flag
- Parallel scanning with 50% faster performance

### Added

- New `order` subcommand to show repos in dependency order (update dependencies first) ([`104f2a2`](https://github.com/grokify/gogit/commit/104f2a2))
- Topological sort using Kahn's algorithm for dependency-aware ordering ([`7f8ff17`](https://github.com/grokify/gogit/commit/7f8ff17))
- Time-based filtering with `-since` flag supporting `7d`, `2w`, `1m` duration formats ([`33af0ef`](https://github.com/grokify/gogit/commit/33af0ef))
- Transitive dependency tracking with `-t` flag to include repos that depend on modified repos ([`7f8ff17`](https://github.com/grokify/gogit/commit/7f8ff17))
- Unpushed commit detection with `-u` flag to filter repos needing push ([`7f8ff17`](https://github.com/grokify/gogit/commit/7f8ff17))
- Internal dependency display showing `(depends on: x, y)` in output ([`33af0ef`](https://github.com/grokify/gogit/commit/33af0ef))
- Latest modification time tracking per repository ([`7f8ff17`](https://github.com/grokify/gogit/commit/7f8ff17))
- Parallel scanning using goroutine worker pool (uses GOMAXPROCS workers) ([`7f8ff17`](https://github.com/grokify/gogit/commit/7f8ff17))
- Lazy evaluation of expensive operations (mod time, unpushed checks) ([`7f8ff17`](https://github.com/grokify/gogit/commit/7f8ff17))
- Aligned output formatting with consistent column widths ([`33af0ef`](https://github.com/grokify/gogit/commit/33af0ef))

### Changed

- Migrated CLI from standard `flag` package to Cobra for better UX ([`33af0ef`](https://github.com/grokify/gogit/commit/33af0ef))
- Added short flags: `-d` (dir), `-f` (format), `-r` (recurse), `-s` (since) ([`33af0ef`](https://github.com/grokify/gogit/commit/33af0ef))
- Output now shows numbered list with alphabetical sorting ([`33af0ef`](https://github.com/grokify/gogit/commit/33af0ef))
- Refactored scanner package with `ScanOptions` for configurable behavior ([`7f8ff17`](https://github.com/grokify/gogit/commit/7f8ff17))

### Performance

- Optimized git status to single command per repo (was 2-3 commands) ([`6e88de7`](https://github.com/grokify/gogit/commit/6e88de7))
- Scan time reduced by ~50% (6.7s → 3.4s for 600 repos) ([`6e88de7`](https://github.com/grokify/gogit/commit/6e88de7))

### Dependencies

- Added `github.com/spf13/cobra` for CLI framework ([`33af0ef`](https://github.com/grokify/gogit/commit/33af0ef))
- Added `github.com/grokify/mogo/fmt/progress` for progress bar rendering ([`9c27c2a`](https://github.com/grokify/gogit/commit/9c27c2a))

### Documentation

- Updated README with order subcommand, new flags, and performance section ([`e319764`](https://github.com/grokify/gogit/commit/e319764))
- Added RELEASE_NOTES_v0.2.0.md ([`e319764`](https://github.com/grokify/gogit/commit/e319764))

## [v0.1.0] - 2026-01-17

### Highlights

- CLI tool to scan multiple Git repositories and identify repos needing attention
- Zero external dependencies - uses only Go standard library

### Added

- Repository scanner that checks for uncommitted changes using `git status --porcelain` ([`97d0739`](https://github.com/grokify/gogit/commit/97d0739))
- Replace directive detection in go.mod files (both single-line and block format) ([`97d0739`](https://github.com/grokify/gogit/commit/97d0739))
- Module name mismatch detection comparing go.mod module path with directory name ([`97d0739`](https://github.com/grokify/gogit/commit/97d0739))
- Dependency filtering with `-dep` flag to find repos depending on a specific module ([`97d0739`](https://github.com/grokify/gogit/commit/97d0739))
- Recursive nested go.mod search with `-recurse` flag for monorepo support ([`97d0739`](https://github.com/grokify/gogit/commit/97d0739))
- Progress bar with percentage, count, and current repo name during scanning ([`97d0739`](https://github.com/grokify/gogit/commit/97d0739))
- Two output formats: human-readable list (default) and compact markdown table ([`97d0739`](https://github.com/grokify/gogit/commit/97d0739))
- Summary statistics showing counts of uncommitted changes, replace directives, and mismatches ([`97d0739`](https://github.com/grokify/gogit/commit/97d0739))
- Tilde expansion for home directory paths ([`97d0739`](https://github.com/grokify/gogit/commit/97d0739))
